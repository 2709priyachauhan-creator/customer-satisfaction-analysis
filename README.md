# customer-satisfaction-analysis
This repository contains an end-to-end data analytics project focused on predicting customer satisfaction ratings using real-world support ticket data. The workflow covers data cleaning, advanced feature engineering (including text and temporal features), exploratory analysis, model selection (Random Forest, XGBoost), hyperparameter tuning, and visual reporting.

### Key features:

Step-by-step Jupyter notebook with clear sections and markdown explanations

Text analytics: TF-IDF, sentiment analysis, keyword/topic flags

Feature engineering: temporal features, interaction, missing value flags, repeat customer identification

Model evaluation and comparison across multiple algorithms

Confusion matrix, feature importance, and error distribution visualizations

Comprehensive error analysis and discussion

### Usage:

Open the notebook (.ipynb file) directly on GitHub to view code, explanation, and results

Clone or download the repository to run analyses locally
